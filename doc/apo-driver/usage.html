1. Select **only the driver** described on the Audio Enhancers page (APO Driver Component) during APO Driver installation.
    • Do not select multiple child components if the parent component specifies "[Choose Only One]." For example:
      • Dolby DAX API Service [Choose Only One] (Parent Component)
        • Dolby DAX API Service 2.3.231.0 (child component)
        • Dolby DAX API Service 3.20403.436.0 (child component)
      • Select either Dolby DAX API Service 2.3.231.0 or Dolby DAX API Service 3.20403.436.0—do not select both.
    • If the message "Service 'UWP PRC Service' failed to start" appears during installation, **uncheck** "UWP Services" under "Sound Blaster Connect."
    • To avoid NahimicService.exe modifying Stream FX and Mode FX registry keys, **deselect** Nahimic driver installation if you do not want Nahimic 3.
  2. Open FX Configurator via Lanuher.exe and select your desired endpoint.
  3. Click the "Backup" button under the "Endpoint Registry" section. Follow the audio enhancer installation instructions.
  4. Use the "Product Config Tool" button under the "Tools" section. Select a product from the dropdown list and click "Apply Product Settings to Selected Endpoints."

 

Additional Notes:

  • To install/remove additional APOs: Go to Start Menu > APO Driver > Maintenance Driver.

Uninstalling APO Driver:

  1. Use the "Restore" button under the "Endpoint Registry" section for any endpoints modified via Product Config Tool.
  2. Close all sound enhancer processes.
  3. Uninstall APO Driver via Windows Programs and Features.

Silent Install:

You can perform a silent installation using the following command:

apo_driver_setup_x64.exe /exenoui /qn ADDLOCAL="Base,Identier,Identier2"

Note: Ensure you include Base in the ADDLOCAL parameter to install required core components.
